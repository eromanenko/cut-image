Instruction: Upload an image or a PDF document. Click to add a horizontal line. Hold Shift + Click for a vertical line. Or use Auto-Detect Cut Marks to find separator lines automatically.

Editing: Hover over a line and hold Shift to drag it. Select a line to move it precisely with Arrow Keys (hold Shift for faster move) or remove it with Delete.

Filtering: Set Min size to exclude small fragments from the archive. Set DPI to match your image resolution for accurate mm conversion.

PDF Pairing: When a multi-page PDF is loaded with All pages checked, you can choose a pairing mode for double-sided cards. Horizontal mirror — the back page is flipped left-to-right (odd pages are fronts, columns are reversed on even pages). Vertical mirror — the back page is flipped top-to-bottom (rows are reversed on even pages). Files are named prefix1a / prefix1b (front/back of the same card), numbered continuously across page pairs. If the PDF has an odd number of pages, the last page is ignored.

Instruction: Upload a scanned image or PDF document containing cards. Click and drag the red corners to adjust borders. You can also drag the whole card by clicking inside it.

Auto-Detect: Specify exact Width and Height (mm) below for highly accurate detection. Use 🎯 Get from 1st Card to auto-fill dimensions.

Navigation: Use / key to cycle through corners. Enter jumps between cards. Arrow Keys (+ Ctrl for ×10 step, + Shift to move opposite corner too).

Zoom Window: + / - to adjust magnification. Z toggles zoom. Rectangle Mode: All cards are identical parallelograms defined by global Width, Height, and Skew (px) values above.

Add card: Click empty area on canvas — a new card appears at the click position. Drag to reposition. Add Manual button centres a card in the viewport.

Move: Drag any card. Arrow Keys for precise 1px steps (Ctrl = 10px steps).

Rotate: Press / (0.05° CW) or Shift+/ (0.05° CCW). Enter / Shift+Enter to jump between cards.

Zoom Window: Shows all 4 corners simultaneously. Align each corner to the yellow frame. + / - to adjust magnification.
